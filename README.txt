Invitación de Cumpleaños - Página Web

Esta es una página web interactiva para una invitación de cumpleaños. La página incluye una cuenta regresiva que muestra cuántos días y horas faltan para el evento, así como detalles sobre la fiesta.

Detalles del Evento

Fecha del Evento: 26 de enero
Hora: 10:00 AM
Lugar: Playa
Vestimenta: Ropa blanca

Características

- Animación de entrada al cargar la página.
- Contador de días y horas que se actualiza automáticamente.
- Estilo "Aesthetic" de playa y temática de cumpleaños.
